Generates and returns the set of points to draw an arc.

Parameters
posThe starting position of the arc.
directionThe direction of the arc, 0 for clockwise, 1 for counter-clockwise.
rThe radius of the circle. A negative value will draw the short arc for the given value, while a positive value will draw the long arc.
penThe state of the pen. 1 for pen down, 5 for pen up.
feedThe feed rate of the operation. Given in inches per minute.
x_relThe x coordinate of the arc's end point.
y_relThe y coordinate of the arc's end point.
